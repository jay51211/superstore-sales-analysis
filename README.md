\# Superstore Sales Analysis



\## Overview

This project performs exploratory data analysis (EDA) on a global superstore dataset using Python. 

The analysis focuses on sales performance, profit distribution, and country-wise insights to support data-driven decisions.



\## Tools Used

\- Python

\- Pandas

\- Matplotlib

\- Jupyter Notebook



\## Key Analysis

\- Total sales and profit metrics

\- Sales analysis by country

\- Profit analysis by market

\- Identification of top-performing regions



\## Repository Structure

\- superstore\_sales\_analysis.ipynb – Main analysis notebook

\- superstore.csv` – Dataset

\- image/ – Visual outputs and screenshots
